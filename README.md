# SIComp: Setup-Independent Projector Compensation

SIComp is a projector compensation benchmark and reproduction codebase for setup-independent compensation across multiple surface settings.

This repository includes:
- inference scripts for the SIComp variants used in the paper,
- baseline comparison scripts,
- evaluation scripts for surrogate and real compensation metrics,
- a one-command reproduction entry script.

## Overview

The main reproduction workflow is implemented in:
- `src/python/reproduce_paper_results.sh`

That script will:
1. run all main SIComp and baseline models on the benchmark datasets,
2. evaluate surrogate metrics,
3. evaluate real compensation metrics,
4. export summary metric tables.

## How to reproduce the paper results

### Step 1: Prepare checkpoints

Make sure the required pretrained weights and checkpoints are available under `checkpoint/` and any model-specific checkpoint directories used by the scripts.

### Step 2: Prepare the dataset

Download the dataset and place it under:

```text
data/SIComp_Benchmark/
```

### Step 3: Run the full reproduction script

From the repository root, go to `src/python` and run:

```bash
cd src/python
bash reproduce_paper_results.sh
```

This script runs all supported model variants over the benchmark datasets and then evaluates the generated results.

## What the reproduction script runs

The script `src/python/reproduce_paper_results.sh` currently runs:

- `SIComp(#surf=1).py`
- `SIComp(#surf=3).py`
- `SIComp(#surf=5_L1+SSIM+DeltaE+LPIPS).py`
- `SIComp(#surf=5_L1+SSIM).py`
- `SIComp(#surf=5_L1+SSIM+DeltaE).py`
- `SIComp(#surf=5_L1+SSIM+LPIPS).py`
- `FF+CompenNeSt.py`
- `FF+CompenNeSt(Origin).py`
- `FF+PANet.py`
- `FF+PANet(Origin).py`
- `calculate_surrogate_metrics.py`
- `calculate_real_compensation_metrics.py`

## Outputs

After the run, the main outputs include metric summary tables generated by the evaluation scripts.

Typical output files include:
- surrogate evaluation summary Excel files under the surrogate dataset root,
- `Actual_Final_Metrics.xlsx` under the real-compensation dataset root.

Depending on the model script, intermediate prediction images and logs may also be saved under dataset-specific output directories.

## Notes

### 1. Dataset path

The reproduction script uses relative dataset paths:

```bash
../../data/SIComp_Benchmark/SetA_Surrogate
../../data/SIComp_Benchmark/SetA_Real_Compensation
```

So it should be run from `src/python`.

### 2. Shell script path

The current `reproduce_paper_results.sh` contains a machine-specific `cd` command:

```bash
cd /h/home/lihaibo/Tutor_file/SIComp/src/python/ || exit
```

If you are using another machine, update that line or replace it with a path relative to your local repository.

### 3. Environment setup

This repository currently does not include a top-level pinned dependency file in `SIComp/`.
Please install the required Python packages according to the imported libraries used by the scripts, such as PyTorch, OpenCV, pandas, LPIPS, and `pytorch-fid`.

### 4. Pre-publication note

The corresponding paper has not been formally published yet, so no citation information is provided in this repository at the moment.
